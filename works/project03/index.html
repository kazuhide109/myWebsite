GIF JOCKEY SET
Work with Kakuya Shiraishi (SystemPrigraming of Sound)
Role: Consept Idea, System & Visual Making
本作品はクラブ空間を想定した、観客参加型のVJシステムです。
本来ビジュアル／ビデオジョッキーはその名の通り、多くの場合、事前に準備した映像素材を曲や演奏に合わせて切り替えるスタイルをとりますが、DJとVJが別々で独立しており、表現が乖離することがしばしば起こります。そこで、本作品では事前素材を用意せず、会場に設置した撮影ブースで観客に撮影してもらうことにより、その場で映像素材を収集し、操作者が素材を切り替える手法を取りました。観客がDJの演出する空間の中で、身体を動かす姿を記録し、VJがそれを素材にしてミックスすること、さらにDJがコントロールディスクの動きと映像の再生コントロール同期させることにより、VJが観客とDJの双方をつなぐ役割を担い、音と映像、そして観客の間に一体感を作り出します。短いクリップの動画をコミカルに動か様相からGIF JOKCEYと称しています。
